HTML BASICS

Introduction

HTML is an abbreviation for HyperText Markup Language. It is used to create web pages by utilising a markup language. The link between web pages is defined by hypertext. To define the text document within the tag that defines the structure of web pages, a markup language is used.

In this tutorial it will go through the layout of a html page and including best practices, problems to look out for, html tag reference with their descriptions.

Layout

HEADER
NAV
CONTENT
SIDEBAR
FOOTER
  • Header: Includes elements relating to website logo, navigation bar and social links.
  • Nav: Navigation bar which has links to different pages on a website.
  • Sidebar: Would include anchor links which skip down to a relevant heading on the page. May also include advertisements.
  • Content: All the page content would be located here. Main point of the page would be here.
  • Footer: A place on a webpage where important and not needed links and information would be placed. Could include links to a policy page for example.

Note: Not all websites will follow a layout like this, this just an example of how one would most likely look like.
