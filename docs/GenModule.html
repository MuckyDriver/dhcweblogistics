Getting Started

Install

To get this module just click here to get the download file (.lua)
Next, you can copy this .lua file into a roblox place which you are working on. Furthermore, move it to "ReplicatedStorage".

local GenModule = require(Module)

About

This module was made for people who want to generate random strings, letters, symbols, filter words, scramble words and more!



Functions

RandomString

RandomString function creates a random string of letters. You also need to pass through a number which is the length of the string you want to generate.

local RandomString = GenModule.RandomString(8)

RandomCaps

RandomCaps function creates a random string of caps from A-Z. This is similar to the RandomString function however uses caps instead.

local RandomCaps = GenModule.RandomCaps(8)

RandomLetters

RandomLetters function creates a random string of letters only. This function will ignore numbers which are used in the RandomString function.

local RandomLetters = GenModule.RandomLetters(8)

RandomLow

RandomLow function creates a string of random lowercase letters using a given length.

local RandomLow = GenModule.RandomLow(8)

RandomSymbols

RandomSymbols function creates a string of random symbols using a given length.

local RandomSymbols = GenModule.RandomSymbols(8)

Custom

Custom function will create a random string using letters which was given plus the given length as well.

local CustomString = GenModule.Custom(8, "HELLO")

Cut

Cut function will cut trim out a piece of text/string from to points. E.g. Start would be the start char and End would be the last char. Between These two points it will remove it and leave the rest.

local String = "Hello World"
GenModule.Cut(String, 4, 6)

BrokenString

BrokenString function will break up a string making it impossible to know what it was before, there is no way to revert it back to normal. This function will replace and put new characters in its place where the orginal string does not exist but a new broken one.

local String = "Hello my name is John!"
GenModule.BrokenString(String) -- Example Result: ihesmmo ma

Filter

Filter function will return the passed string into hashtags or a custom method.

GenModule.Filter("String", "*")

ScrambleLetters

ScrambleLetters function will scramble up the given string to something else. You can unscramble this as well using another function by passing through the scrambled letters/words string.

GenModule.ScrambleLetters("I'm scrambled for time!") -- d!b am mrsico'rfeel Itm

ScrambleWords

ScrambleWords function will scramble up the given string to something else. On the other hand this function will move the words around instead of all the letters/chars around.

GenModule.ScrambleWords("I'm scrambled for time!") -- I'm for time! scrambled

UnScramble

UnScramble function will revert the scrambled string back to its orignal form.

GenModule.UnScramble("I'm for time! scrambled") -- I'm scrambled for time!
GenModule.UnScramble("d!b am mrsico'rfeel Itm") -- I'm scrambled for time!

RemoveCharInString

RemoveCharInString function will remove a given character away from the string. E.g Apple -> p -> Ale

GenModule.RemoveCharInString("Have a good day!", "a") -- Hve good dy!
