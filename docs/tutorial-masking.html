What is a mask?

A mask would be a ui element which looks like a rectange with a hole in it, you are able to change the transparency of the mask and the color. In addition to this you can also change how the hole design looks like including size, shape and color.



Tutorial

Step 1: Add UI Elements

The UI elements you will need to create would be ScreenGui, A Frame, UICorner, UIScale and UIStroke. These are all needed to create a perfect mask!



Mask is the UIStroke element, HoleSize would be the UIScale element and Roundness would be the UICorner element. All of these elements would be parented to the ScreenGui. The ScreenGui should have IgnoreGuiInset turn on for it to work best.

Step 2: Design your mask

Firstly, designing the MaskLayout frame, this should be pretty easy. You can set the Size property to {0, 100, 0, 100} this will be the normal size of the hole of the mask. As well as size, you can also change the AnchorPoint property to {0.5, 0.5} this will keep it in the middle of objects and better position as a whole.



Secondly, you need to configure the UIScale scale property to whatever you want it to be. This would be the size of the hole in the mask.

Next, the UICorner can be used to turn the mask hole from a rectangle to a circle depending on what you want it to look like.



UIStroke element will need some properties changed to make the mask to work properly. Firstly, you will need to change the thickness to something really big like 99999 or more; this will ensure that you do not see the edge of the stroke of the mask (it will look weird otherwise).
