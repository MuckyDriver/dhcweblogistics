Message from creator

Hello, I am the creator of this project Claasgreeneye or alternately known as DHCGamer1055! Mucky Driver is my biggest Roblox project so far - this project has been going from the end of 2019 until to the present day. I started this project when the first edition of this game "Mucky Driver Sim 19" back in early 2019 became a flop and had loads of improvements to be made. Throughout the development near the start I did not plan it how most people plan a video game, I just thought of it and wrote the main details of the game onto a Google Docs document and started straight away with the map.
